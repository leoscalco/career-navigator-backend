# Career Navigator Logo

## Design Concept

The Career Navigator logo features a **compass** design that symbolizes:

- **Navigation & Guidance**: The compass represents direction and guidance in career decisions
- **Career Growth**: The arrow points upward, symbolizing upward career progression
- **Professionalism**: Clean, modern design with professional color scheme

## Color Palette

- **Primary Blue**: `#2962FF` - Represents trust, professionalism, and technology
- **Accent Orange**: `#FF9800` - Represents energy, growth, and opportunity
- **Dark Blue**: `#193C9B` - Used for text and outlines, adds depth
- **White**: `#FFFFFF` - Used for contrast and clarity

## Logo Files

### Full Logo (with text)
- `logo_1024.png` - High resolution (1024x1024) - For app stores, presentations
- `logo_512.png` - Standard resolution (512x512) - For web headers, documents
- `logo_256.png` - Medium resolution (256x256) - For web use
- `logo_128.png` - Small resolution (128x128) - For smaller displays
- `logo_64.png` - Tiny resolution (64x64) - For favicons, small icons

### Icon Only (no text)
- `icon_256.png` - Large icon (256x256)
- `icon_128.png` - Medium icon (128x128)
- `icon_64.png` - Small icon (64x64)
- `icon_32.png` - Tiny icon (32x32)

### Vector Format
- `logo.svg` - Scalable vector format - Use for print, web (scales to any size)

### Favicon
- `favicon.ico` - Multi-size favicon (16x16, 32x32, 48x48) - For browser tabs

## Usage Guidelines

### When to Use Full Logo
- Website headers
- Documentation
- Presentations
- Marketing materials
- App store listings

### When to Use Icon Only
- Mobile app icons
- Browser favicons
- Social media profile pictures
- Small spaces where text would be illegible

### When to Use SVG
- Print materials (business cards, letterheads)
- High-resolution displays
- When you need to scale to any size
- Web use (better performance than PNG for large sizes)

## Regenerating Logos

To regenerate all logo files:

```bash
poetry run python scripts/generate_logo.py
```

This will create all PNG sizes and the favicon in the `assets/logo/` directory.

## Brand Guidelines

- **Minimum Size**: Logo should not be displayed smaller than 64x64 pixels
- **Clear Space**: Maintain at least 20% of logo height as clear space around the logo
- **Background**: Logo works on both light and dark backgrounds (transparent PNG)
- **Color Variations**: Use the provided color palette. Avoid altering colors without brand approval.
